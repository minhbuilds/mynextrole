# mynextrole
A lightweight, privacy-first job search tracker built as a single HTML file. No login, no database, no subscription. Everything saves locally in your browser.

**#Features:**

Track every application with company, role, recruiter contact, referral, salary, status, and notes
Update application status in one click through Wishlist, Applied, Screening, Interview, Offer, and Declined
Log every interview round separately with date, time, format, interviewer names, notes, and outcome
Company and role fields in the interview log auto-populate from your saved applications
Priority indicators so you can visually sort what matters most
Activity log with a full timestamped history of every status change and update
Export your entire pipeline to CSV anytime
Works offline

**How to use**
Download the file and open it in any browser. Or visit the hosted version at minhbuilds.github.io/mynextrole.
Built with
HTML, CSS, and vanilla JavaScript. No frameworks, no dependencies, no tracking.
